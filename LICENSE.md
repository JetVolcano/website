## 1. Definitions

- **Owner(s):**\
&emsp;The organization or person that owns of the product.
- **Software:**\
&emsp;The product.
- **Dependent:**\
&emsp;Any product that depends on any version of the Software.
- **Fork**\
&emsp;A copy of the Software that is still linked to the software and used for making changes in the codebase.
- **Forking**\
&emsp;The act of creating a Fork and/or modifying it.

## 2. Terms

### What's allowed

- Using/Modifying the Software for private use.
- Forking the Software to contribute to the Software.
- Using the Software as a dependency (if applicable).
- Using the Software as a learning experience.
- Taking inspiration from the Software.
- Selling a Dependent of the Software.

### What isn't allowed

- Publishing/Distrubuting a copy of the Software without proof of the Owner's permission is prohibited.
- Selling a Fork of the Software or the Software itself is prohibited
- Modifying the Software with malicious intent is prohibited.
